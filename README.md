# Spylt

## Introduction

This project demonstrates how to leverage GSAP's powerful animation capabilities to craft fluid transitions, captivating scroll effects, and dynamic UI interactions, combining it with React and Tailwind CSS for a truly immersive and visually stunning website.

## Tech Stack

- ⚛️ React 19
- 🌀 Tailwind CSS v4
- 🎞️ GSAP (GreenSock Animation Platform)

## Features

- ✨ Parallax
- ⚡️ Clip-Path Magic
- 🕹️ ScrollTrigger & ScrollSmoother
- 👏 GSAP Timelines that Actually Feel Good
- 📱 Fully responsive and mobile-friendly

## Quick Start

```bash
# 1. Clone the repo
git clone [https://github.com/KievDev/Spylt.git](https://github.com/KievDev/Spylt.git)

# 2. Install dependencies
npm install
# or
yarn

# 3. Start the dev server
npm run dev
# or
yarn dev
```
